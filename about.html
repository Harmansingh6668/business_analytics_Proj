About ClusterApp

ClusterApp is a web application designed to perform K-Means Clustering on your datasets. Visualize your data and group similar data points easily.

K-Means Clustering

K-Means is an unsupervised learning algorithm that partitions data into K distinct clusters. ClusterApp automates this process and generates interactive scatter plots for analysis.

Data Visualization

Each cluster is visualized using scatter plots. You can view relationships between features, compare clusters, and generate PDF reports for documentation or presentation.
