Project ReadMe

Everything you need to know about ClusterApp and how to use it for data clustering.

Project Overview

ClusterApp is a web application designed to perform K-Means Clustering on datasets. Upload your data, visualize clusters, and generate PDF reports effortlessly. It is built using Python, Flask, Pandas, Matplotlib, Scikit-learn, and ReportLab.

Key Features

  • Upload custom datasets (CSV, TXT, XLSX)
  • Perform K-Means Clustering with adjustable cluster numbers
  • Automatic removal of non-numeric data and NaN values
  • Interactive scatter plots for cluster visualization
  • Download PDF reports containing all plots
  • Future support for DBSCAN and Hierarchical Clustering

How to Use

  1. Go to Start Clustering page.
  2. Select the number of clusters you want.
  3. Choose K-Means as algorithm.
  4. Upload a dataset or choose a predefined dataset.
  5. Click Generate Clusters to view results and plots.
  6. Optionally, download the PDF report.

Technology Stack

  • Python 3.12+
  • Flask (Web Framework)
  • Pandas (Data Handling)
  • NumPy (Numeric Computation)
  • Scikit-learn (Machine Learning)
  • Matplotlib (Plots & Visualization)
  • ReportLab (PDF Generation)
  • Bootstrap 5 (Front-End Design)
